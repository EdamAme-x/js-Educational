`console.log(1 + 2)`

引き算

掛け算
`console.log(5 * 2)`
これで10が出る
×の代わりにプログラムでは`*`(スターと読む)を使う

割り算
`console.log(10 / 2)`
これで5が出る
÷の代わりにプログラムでは`/`を使う

余剰(あまり)
`cosnole.log(10 % 3)`
これで1が出る
